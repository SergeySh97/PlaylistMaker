                                                               # Playlist Maker

An app for searching music and creating playlists on base iTunes API


<details>
<summary>Tools and Technologies Used</summary>

| Technologies                 | Technologies                 |
|------------------------------|------------------------------|
| MVVM                         | Kotlin Coroutines            |
| Single Activity              | Fragment                     |
| Koin                         | Room                         |
| Retrofit2                    | ViewPager2                   |
| MediaPlayer                  | Glide                        |
| RecyclerView                 | SharedPreferences            |
| Gson                         | TabLayout                    |
| LiveData                     | Jetpack Navigation Component |
| BottomNavigationView         |                              |

</details>

<details>
<summary>App Features</summary>

| Features                                       |
|------------------------------------------------|
| Create and delete playlists                    |
| Search for audio tracks                        |
| Listen to a preview of the found track         |
| Edit playlist information                      |
| Save search history                            |
| Add/remove tracks to/from a playlist           |
| Add/remove tracks to/from the "Favorites" list |
| Change the app theme                           |
| Share playlists                                |

</details>
